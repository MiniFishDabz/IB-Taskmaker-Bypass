# IB-Taskmaker-Bypass

A free userscript that allows for full access to the IB Taskmaker database in its entirety.

---

## 🚀 Features
* **Full Access:** Unlocks questions across the IB Taskmaker database.
* **Lightweight:** Runs seamlessly via standard browser extensions.

---

## 🛠️ Installation & Usage

Follow these steps to get started:

1. **Install a Userscript Manager**
   * Download and install a userscript manager extension for your browser, such as [Tampermonkey](https://www.tampermonkey.net/) or Violentmonkey.

2. **Install the Script**
   * Navigate to the `main.user.js` file within this repository.
   * Click on the **Raw** button in the top-right corner of the file view.
   * Your userscript manager should automatically prompt you to install the script. Click **Install**.

3. **Access the Database**
   * Go to the official IB Taskmaker website. The content should now be available.
